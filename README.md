## 🛠️ Getting Started

### 🟦 Clone this repository

```bash
git clone https://github.com/JlBestMc/Test-Level-Fizz-Buzz.git

```

### 🟩 Install Dependencies

Make sure you have Node.js installed. Then install the packages:

```bash
npm install
```
